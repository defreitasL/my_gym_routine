# My Gym Routine

Interactive mobile-first workout and nutrition website for a 4-day Upper/Lower routine.

Built as a static site for GitHub Pages.

## Files

- `index.html` — app shell and page structure
- `style.css` — responsive styling
- `app.js` — workout data, routing, checkboxes, progress tracking and diet page rendering

## GitHub Pages

In the repository settings, enable GitHub Pages from the `main` branch and root folder.
